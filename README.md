# PrimeTrade.ai — Trader Behavior Insights
## Market Sentiment vs Trader Performance Analysis

This repository contains the completed data science assignment for the Junior Data Scientist role at PrimeTrade.ai.

### Project Overview
The project analyzes the relationship between Bitcoin market sentiment (Fear & Greed Index) and trader performance on Hyperliquid. It involves data cleaning, merging disparate datasets, exploratory data analysis, and advanced statistical modeling to uncover actionable trading insights.

### Contents
- `PrimeTrade_Analysis.ipynb`: The primary analysis notebook, fully executed.
- `fear_greed.csv`: Bitcoin Market Sentiment dataset.
- `historical_data.csv`: Historical trader performance data from Hyperliquid.
- `chart_*.png`: High-resolution visualizations generated during analysis.

### Key Insights
1. **Contrarian Edge**: Traders achieve higher average PnL during "Fear" phases.
2. **Top Trader Profiling**: Elite traders (top 10%) significantly reduce exposure during "Extreme Greed" compared to the average trader.
3. **Sentiment Transitions**: Regime shifts (e.g., Fear → Neutral) are identified as high-alpha trading opportunities.
4. **Volume vs Profit**: High trade volume during "Greed" phases often correlates with compressed margins due to fee drag and crowded positioning.
